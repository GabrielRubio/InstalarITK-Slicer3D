Instalação:

Instale o VTK antes do SLicer!
->instalado CMAKE sudo apt-get install cmake
-> instalado openGL sudo apt-get install freeglut3-dev
-> cria-se a pasta general_source mkdir general_source cd general_source
-> na pasta general_source cria-se a pasta VTK mkdir VTK cd VTK
-> criando as pastas de source e build mkdir source mkdir build
-> na pasta de source clonando o vtk 7.1.1 git clone git://vtk.org/VTK.git ou baixar os arquivos do site
-> na pasta de build usando cmake cria os arquivos de build cmake ~/VTK/source/ ou usa o cmake gui
-> No Cmake, limpa o cache; acha a pasta ~/vtk/source e ~/vtk/build usa configure e generate
-> na pasta build dar make make [-j3]
-> na pasta general_source cria-se a pasta ITK mkdir ITK cd ITK
-> criando as pastas de source e build mkdir source mkdir build
-> na pasta de source clonando o itk 4.12.1 ou baixando os arquivos do site
-> na pasta de build usando cmake cria os arquivos de build cmake ~/VTK/source/ ou usa o cmake gui
-> No Cmake, limpa o cache; acha a pasta ~/itk/source e ~/itk/build usa configure e Adiciona as variáveis: ====> Module_ITKVtkGlue ON ====> VTK_DIR /opt/compilation/paraview-git_build/VTK/ configure novamente e generate
-> na pasta build dar make make [-j3]
-> na pasta build dar make install sudo make install
--> instalando slicer
--> instalar usando o site: https://www.slicer.org/wiki/Documentation/Nightly/Developers/Build_Instructions#Ubuntu
